The Vision

What if a text adventure had a real combat engine? Not simplified stat checks or coin-flip RNG — a complete d20 system with six attributes, base attack bonuses, armor class, feats, saving throws, and a three-action economy. Every attack rolls a real d20. Every hit location matters. Every wound has mechanical consequences.

Under the surface, it's d20 + BAB + STR vs AC. You never see the math. You never see the dice. You see prose — authored, polished, and woven into a living narrative. Your input sits quietly at the bottom of the screen. The narrative view reads like a book, with your actions blended seamlessly into the author's prose. The mechanics are Pathfinder. The experience is a novel you're writing.

Authored content enhanced by AI — not generated by it. Every room, every NPC, every dialogue tree is hand-crafted. LLM narration polishes dramatic moments — composing combat poetry, threading scene descriptions, varying phrasing so repetition never breaks immersion. If the AI is unavailable, template fallback keeps the game running perfectly. This isn't AI improvisation. It's AI composition.

Key Innovations

>

Natural Language Commands

Type what you'd naturally say. Pattern matching parses commands at 0ms. LLM classification handles ambiguous input as a fallback — no rigid parser vocabulary to memorize.

d20

Real Pathfinder Mechanics

Full PF2e three-action economy with multiple attack penalty. Six attributes, base attack bonus, armor class, feats, saving throws. The narrative reads as prose — but the action panel lets you watch the dice roll in real time.

✦

LLM-Enhanced Narrative

Templates render instantly. LLM narration polishes dramatic moments — combat poetry, scene composition, varied phrasing. Template fallback means nothing breaks if AI is unavailable.

◆

Graph-Based Dynamic World

Kuzu graph database as single source of truth. Five condition types gate every exit, dialogue, and quest trigger. Persistent topology tracks every entity and relationship in real time.

❞

Professional Ink Dialogue

The same narrative scripting language behind 80 Days and Heaven's Vault. Authored branching dialogue with state tracking, conditional responses, and deep character development.

⚔

Wound-Location Combat

Attacks target specific body locations. Severity tiers from scratched to destroyed. Limb damage applies real mechanical penalties — a wounded sword arm means worse attacks.

{}

Data-Driven Everything

All game logic lives in JSON. Object interactions, NPC conditions, quest triggers, combat templates — change data, change behavior. Zero hardcoded special cases.

WorldBuilder

A macOS companion app for world creation.

WorldBuilder is a visual graph editor for designing RPGFuture worlds. Build locations, connect exits, define NPCs and their dialogue trees, place objects, design quests, and author puzzles — all in a node-based interface that maps directly to the game's graph database.

  • Visual graph editor with drag-and-drop entity placement
  • Entity inspectors for every game type — characters, objects, locations, quests, puzzles
  • AI-assisted content generation for descriptions, dialogue, and flavor text
  • 10-layer authoring stack — from world skeleton to compiled Ink dialogue
  • Import and export complete worlds as validated JSON
